# Ledger Accounting Tests

This project contains Playwright end-to-end tests for ledger accounting functionality.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Install Playwright browsers:
   ```bash
   npx playwright install
   ```

3. Run the tests:
   ```bash
   npx playwright test
   ```

4. View the HTML report:
   ```bash
   npx playwright show-report
   ```

## Project Structure

- `tests/` - Contains the test files
- `playwright.config.ts` - Playwright configuration
- `package.json` - Project dependencies and scripts

## Writing Tests

Tests are written using Playwright's test runner. See the [Playwright documentation](https://playwright.dev/docs/writing-tests) for more information.
